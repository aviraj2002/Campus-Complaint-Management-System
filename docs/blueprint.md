# **App Name**: CollegeComplain

## Core Features:

- Create Complaint: Allow students to create a complaint ticket including title, description, category and priority.
- View Own Complaints: Allow students to view the status of their own complaints.
- View All Complaints: Allow admins to view all complaints.
- Assign Staff: Allow admins to assign staff to handle a complaint.
- Update Status: Allow staff and admins to update the status of a complaint.
- User Authentication: Secure the app using Firebase Authentication to verify users.
- Role-Based Access Control: Implement role-based access control to protect the system and control student/admin/staff user permissions.

## Style Guidelines:

- Primary color: Navy blue (#1E3A8A) to evoke trust and reliability.
- Background color: Light gray (#F5F5F5) for a clean, professional look.
- Accent color: Teal (#008080) for important actions and highlights.
- Body font: 'Inter', a grotesque-style sans-serif, for a modern, objective feel.
- Headline font: 'Space Grotesk', for headlines.
- Use consistent icons to represent complaint categories and status.
- Maintain a clean and organized layout for easy navigation.
